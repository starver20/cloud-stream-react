#CloudStream

## **Tech Stack**

- React
- React Context API
- React router
- CloudUI

## **Key Features**

- Auth

  - Login
  - Logout
  - Signup

- Filter Videos By

  - Categories

- Playlist

  - Create new playlist
  - Add videos to playlist
  - Remove videos from playlist
  - Delete playlist

- History

  - Add to history
  - Remove from history
  - Clear history

- Watchlater

  - Add to watchlater
  - Remove from watchlater

- Likes

  - Add to Likes
  - Remove from Likes

- Single video page
